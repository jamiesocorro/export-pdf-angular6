# ExportToPDF


## Install

```bash
$ git clone https://github.com/jamiesocorro/export-pdf-angular6.git
$ cd export-pdf-angular6
```

## Run Application
```bash
$ npm i
$ npm i rxjs
$ npm i jsPDF
$ npm i jquery
$ npm start
```

Once the application starts it will be available in 
`http://localhost:4200`
